Temporary blog system

I started to create a publishing system based on node.js. It's a static site generator. But unlike most of the static site generator, it's intended to be used not only by geeks but by any Web publisher. It doesn't have a name yet, I call it web-writer for now. I'll make it public on GitHub as soon has I have something functional.

Since I want a blog right now and the web-writer isn't ready yet, I just wrote this blog in plain HTML using Twitter's Bootstrap and hosted using GitHub Pages. At same time, I hope that starting to publish content myself will help me understand better what my publishing system should be.
