RepRap - Electronics

I've been working for a while on building a 3D printer. I'm building de Prusa Mendel from the RepRap project. When it come to the electronic you have a lot of options. But most of them are pretty similar. They have an ATmega microprocessor and Pololu stepper drivers. A already have an Arduino Uno which is based on an ATmega328, so I decided to base my electronic on this microprocessor. The problem is most RepRap electronic is based on an ATmega1284 or an ATmega2560. The main differences with the ATmega328 is that it is has less memory and less I/O pins. According to many people the memory part is not an issue. For the I/O pins what it means is that it probably won't support dual extrusion or a heat bed.

So I based my electronic on the RAMPS v1.4, but I didn't kept their board layout.

Temporary blog system

I started to create a publishing system based on node.js. It's a static site generator. But unlike most of the static site generator, it's intended to be used not only by geeks but by any Web publisher. It doesn't have a name yet, I call it web-writer for now. I'll make it public on GitHub as soon has I have something functional.

Since I want a blog right now and the web-writer isn't ready yet, I just wrote this blog in plain HTML using Twitter's Bootstrap and hosted it using GitHub Pages. At the same time, I hope that starting to publish content myself will help me understand better what my publishing system should be.

The blog code is hosted at github.com/fgascon/fgascon.github.com.
